################################ 
Configurer la base de données###
################################
 
1 Dans notre projet, nous avons installé et configuré l'application "phpmyadmin", 

donc vous pouvez vous connecter à la base de données à partir de http://127.0.0.1/phpmyadmin.

2 : Créer une base de données appelée "geonote" et importer les tables en utilisant le fichier "geonote.sql"
 
3: Vérifier que le login  est : "root" et mot de passe est : "root".




##############################
###Configurer l’application###
##############################
 
Pour lancer l'application , il suffit de lancer le projet Geonote sur l'IDE de NeatBeans.

Pour se connecter en mode qu'administrateur, le login est Admin et le mot de passe est Admin aussi
Pour se connecter en mode client , le login est Demo et le mode de passe est Demo aussi.

L'inscription des nouveau utilisateur n'est pas disponible et n'est pas demandé dans le cahier des charges.
